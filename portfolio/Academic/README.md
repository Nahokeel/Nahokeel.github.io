# Academic Projects

Put your school-related projects in this folder.

## Suggested structure per project

Create one folder per project:

- `project-name/`
  - `icon.svg` (project icon)
  - `preview.gif` or `preview.mp4` (demo)
  - `README.md` (project details + links)

## Media tips

- Use GIF for short quick previews.
- Use MP4/WebM for better quality and smaller size.
- Keep file names simple (lowercase, hyphenated).
