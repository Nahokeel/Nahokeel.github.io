# Portfolio Starter (GitHub Pages)

A clean starter portfolio site built with plain HTML, CSS, and JavaScript.

## Files

- `index.html`: Page structure and content
- `styles.css`: Colors, layout, typography, responsive styles
- `script.js`: Mobile menu toggle and dynamic copyright year

## Run locally

Open `index.html` in your browser.

## Customize quickly

1. Replace name, headline, and about text in `index.html`.
2. Update the project cards with your real work.
3. Set your real email, GitHub, and LinkedIn links.
4. Tweak colors in `styles.css` under `:root`.

## Publish to GitHub Pages

1. Create a GitHub repo named:
   - `<your-username>.github.io` for a user site, or
   - any repo name for a project site.
2. Push these files to the repo.
3. On GitHub, go to Settings > Pages.
4. Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: `main` and `/ (root)`
5. Save and wait ~1-2 minutes.
6. Open your live URL shown in Pages settings.

## Tips for a strong portfolio

- Keep project descriptions outcome-focused.
- Include links to source code and live demos.
- Prefer 2-4 excellent projects over many weak ones.
- Make sure contact links are correct and easy to find.
